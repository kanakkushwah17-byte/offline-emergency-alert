Offline Emergency Alert System

A simple student project using HTML, CSS, JavaScript, Flask, and SQLite.

Project Overview

This system allows a user to send an emergency alert message without using any external online service. Each alert is stored inside a local SQLite database and displayed on the screen.

Frontend

HTML for structure, CSS for design, and JavaScript for page navigation.

Backend

Python Flask handles form submission, routing, and database operations.

Database

SQLite stores emergency alerts locally in a simple and lightweight way.

System Architecture

Step 1: User enters name and emergency message in the form.

Step 2: Frontend sends data to Flask backend using POST request.

Step 3: Flask saves the alert in SQLite database.

Step 4: Stored alerts are fetched from database.

Step 5: Alerts are displayed back on the web page.
